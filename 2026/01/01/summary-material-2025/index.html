2025年画材使用记录

Posted on 2026-01-01 In 绘画相关 Views:

与往年不同，今年算是正式入了水彩的坑，开启了探索手绘材料的旅途。虽然每次看画材测评都总觉得逃不开某种消费主义的味儿，但本着学习记录的目的，还是写下了这篇小结。Disclaimer: 没有进行比较严谨的测试，纯主观感受记录，请根据自身需求理性消费。

颜料类(Paint)

水彩(Watercolor)

Chromatek Watercolor

于2022年2月受赠于友人及同事，算是我在北美这边拥有的第一套水彩颜料，在今年开始尝试水彩后终于派上用场。虽然感动于友人同事的相赠之情，然而用久后发现色料不好，颗粒很大，画起来有搓泥之感，很难控制。遂出。

目前在我的小分装盒里还剩一些，第一本迷你速写本主要就是用这套颜料画的，也不算浪费了友人同事的好意。

Winsor Newton - Cotman Watercolor 20 Color Paint Set

于2025年4月购于亚马逊。本人的第一套温莎牛顿，在今年也算是我的主力颜料。颜色清新、颗粒细腻，价格亲民，比起DS, MG等这边动辄几十上百刀的贵价套装，二十来刀的Cotman对萌新来说显得格外具有性价比。除了因为是学生级颜料色料不够浓郁之外，个人觉得没有什么大缺点。淘宝上买单只就更便宜了（8ml大概8-10块钱一只），除了颜色不够多，缺单色素颜料之外，可谓是便宜大碗好用，强烈推荐给新手朋友。

基本上我上水彩课的作业、平常的练习都是用这套颜料画的。最开始我是挤一坨在调色盘，如果干了的话就在用之前喷点水再蘸取，但发现这样颜料浓度不太好控制，有时已经蘸了好多次了但颜色还是淡淡的，后来就改成用管子现挤现用了。总的来说，也算是四平八稳没啥大缺点的一套颜料。

Winsor Newton - Cotman Watercolor Field Box Set of 12, Half Pans

于2025年11月购于亚马逊。出于升级写生工具的目的（实在不想再用Chromatek）+看上了这个盒子的设计才买的。不过实际使用下来却没有想的那么方便，如果带了便携喷壶以及自来水笔，那它自带的水壶、洗笔还有小笔感觉也没太大用处；可能还是比较适合带着坐在某个地方画画。而且默认配色我个人不是特别喜欢，尤其是沙普绿，感觉似乎比其他的颜色颗粒要大；后期自己应该还会做些调整。

年底旅游的时候带着这盒去写生了，表现比Chromatek还是要好上不上。只可惜我的速写本在旅途的时候被我弄丢了，等明年画完新本再来放图了。

Holbein Watercolor 24 color set

于2025年11月购于淘宝（海运12月拿到）。试色的时候感觉就是色彩明艳，颜料细腻，因为不怎么扩散所以不是很难控制。在办公室摸鱼随手涂鸦的时候，发现尽管很多不是单色素颜料，但不管怎么瞎混色，色彩都不会很浑浊，到纸面上也不会很难看（这就是金钱艺术家级颜料的力量吗.jpg)。不过因为买的分装，所以感觉可能还没发挥到这套颜料的真正实力；而且现在手上的温莎还没用完，也没太想清楚可以用这套画什么。

不透明水彩(Gouache)

Nicker 36 color set

于2025年5月购于淘宝，本人的第一套不透明水彩分装。一开始用起来没啥特别的感觉，直到和下文将要讲到的Arteza对比起来才发现它的好：细腻不搓泥，混色铺色覆盖都没问题。当然分装颜料会干裂，也有人吐槽说干了后难蘸取，但是我也没吃过新鲜管状Gouache的细糠，所以也不知道怎么评价。可能等用完这盒之后会去买盒管状Gouache试试。

Arteza Gouache Paint Set, 60 pc (?)

于2025年11月受赠于一位出二手的老太太。老太太本人已经不记得这是哪个牌子的颜料，一开始也以为是水彩。经考证，个人觉得可能是Arteza Gouache 60 sets（虽然手上只有48色）。收到这盒颜料的时候里面的颜料已经干透，尝试用喷水+戳碎+搅拌的方式复活。但不知道是颜料本身的特性还是我处理方式的问题，感觉这套颜料画起来又搓泥又油腻，然后还不太铺得开，常常笔上调色盘上已经蘸了很多，但到纸上画两笔就没了。不过这套颜料的配色还是深得我心，非常喜欢里面的橙粉色系和绿色系，用来无脑涂鸦和画童插风格的图都很合适（但不推荐专门花钱买）。

纸/本子(Paper/Sketchbook)

Canson XL Watercolor Pad

一共有两本，一本是不知道什么时候在walmart买的线圈本，一本是和同事的水彩套装附赠的Pad。都是木浆纸，前期的时候用来涂鸦试色，在去上水彩课画湿画法时发现其表现着实拉跨，遂听劝购入棉浆纸。现在主要是调色时用来试色，其他的用法有待开发中。

Arches

听劝购入的阿诗。一般是上课画作业用。

用完一圈别的纸回来后发现确实一分钱一分货；阿诗的吸水性、显色、容错率都比其他纸高上不少。可能新一年会蹲蹲deal，争取在不让钱包太受伤的情况下多买一些吧。

宝虹 - 中粗 四面封胶本

因为实在舍不得用阿诗的纸来画练习和临摹（太贵），遂于淘宝购入宝虹水彩纸。一般用来练习、临摹等，都是两面一起用。

一小部分是临摹一三菌落的画（方块是临摹、侧边有些自己画的）。

一部分用来画风景（自己拍的照片和别人的图都有）。

还有几张是从Santa Fe回来后，受奥基芙启发画的植物花卉。

没啥特别的感受，便宜大碗画就完了。

遵爵 - 方形水彩本

一本是大号的16x16cm的细纹本，一本是6.5x6.5cm的小号口袋本。

大本我主要用来画静物，在今年秋天的时候启用，画了许多的叶子。个人很喜欢这个尺寸和细纹的质感，但是不知道是所在地气候干燥还是这个纸的原因，每次想画湿画法的时候纸都干得巨快，往往铺好一层还没来得及混色纸就干了，很容易留水痕。还有一个这类本子的通病，就是画跨页的时候如果不小心水多了，水和颜料会顺着中缝渗透到其他页面，很容易造成染色惨案，须小心使用。

小本则是日常带出去写生用。个人非常喜欢这个尺寸的小本。揣进兜里，带上迷你调色盒和自来水笔，即可实现随地大小画。不过缺点就是不太经造，第一本书脊的部分在我画完前就已经有脱落之势，纸的质量估计也一般；但看在价格的份上可以接受，已回购。

关于用水彩写生这件事，说来也有意思，我最开始买iPad的原因之一就是为了写生，当时想的是有了iPad就可以画色彩且不用带别的工具了；然而实际使用下来发现自己其实并不怎么喜欢用iPad来画画，做做color recording和摸鱼顶天了。反倒是之前我以为会很麻烦的水彩能让我放松自在地画，口袋揣个小本小盒自来水笔也比扛个iPad要轻松，想来也是很有趣的一个体验。

遵爵 - 卡片水彩纸

买画材给的赠品。一开始觉得纸不是很好所以主要用来瞎画，后面发现这个尺寸带出去也挺方便的。而且这个铁圈的设计可以让你写生画完后拎着本子边走边晾干，好评。


米范儿 180g Sketchbook

淘宝上买的杂牌(?)速写本，本来一开始是想用来画彩铅或者油画棒的，然而今年对干性材料实在是兴趣缺缺，后来就用来画不透明水彩了。刚拿到手的时候也是有一股化学气味，需要晾几天，180g的纸勉强能用来画gouache，但是水加多涂多了的话纸还是会皱，总的来说用来涂鸦和试画材还行，不太适合正经练习和作品。

打印纸/信封

在清理颜料的时候顺手拿过来画的，属于边角料循环再利用。有几张个人还蛮喜欢的，也是个试色彩搭配的好办法。

笔

杂牌画笔

基本购于Temu，亚马逊或者之前同事赠送的chromake水彩套装里带的。Temu买的2刀的画笔套装已然是主力画笔，除了个别笔不聚锋和偶尔笔头会掉落之外，也算对得起它的价格。亚马逊上买的是Mr.Pen的自来水笔，除了挤的时候水出得有点多外还算勉强能用；后面买了吴竹的自来水笔就主要用这个了。Chromake送的自来水笔则是非常难用，有时拼尽全力都挤不出水来，现在基本用它来做一些不愿用其他笔来干的脏活（涮洗笔桶，试可能伤笔的画材等等）。

开始海淘后也开始购入一些单只画笔，目前还在等快递中。

阿泰诗126色彩铅

因为霹雳马太贵且当时不确定自己会用多久彩铅而入的一套，现在看来是个正确的决定。不过一分钱一分货，这套彩铅刚打开时化学味很重，即便已经放在外面晾了好几天，后续使用的时候还是经常能闻到刺鼻的味道。除此之外目前轻度使用没什么大问题，我一般会配合水彩使用，来加重线条或者做质感；也尝试过单用彩铅画画，然而着实废手，遂弃。

总结

算账环节

本来想拉个明细清单和库存状态的，但是时间实在太赶了于是作罢。估算下来还是颜料和纸最花钱。在亚马逊买的两盒温莎牛顿、阿诗的纸以及调色盘留白液，大概花了90来刀；然后还在本地收了一些二手画材，估计总价20刀左右；而海运则主要是颜料分装、单只/固体颜料、彩铅和其他画材、本子、笔等等，花销大概在900元左右，可能加上邮费大概在1000元出头。虽然海运的花费看起来比较高，然而同样的钱，我用海运能买到的东西，要远比在当地能买到的东西多得多；还是感谢万能的国内电商和华人海运，也是让孩子享受到国内先进的制造业了。

虽然但是，感觉在材料上花的钱还是多少有点出乎我的意料，算之前没想到会到这个数（想想板绘真是便宜又环保……十年前买的板子现在还在用）；新一年在把手头的画材用得差不多之前，应该暂时不会再购入新的了。

结语

总的来说，感觉画材这种东西，就是没有对比就没有伤害；没用到过好的就不知道现在用的有多烂。虽然材料的特性要在对比和使用中才能感受到，对材料的学习(maimaimai)也是没有止境的，但如果把画面效果全都归因于材料，又有些本末倒置之感。总之新的一年，还是希望能够多画少买，还是把心思多放在画画本身上。
